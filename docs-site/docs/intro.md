# MailQueue.net documentation

MailQueue.net provides a pragmatic, hosted workflow for queuing and sending email.

## Origins

MailQueue.net is derived from **MailQueueNet** by Daniel Cohen Gindi:
https://github.com/danielgindi/MailQueueNet

The original project provided a solid foundation for queueing and processing email workloads, and this documentation builds on those concepts for the MailQueue.net platform.
